## Renderer

Install the `tegaki` package to render handwriting animations in React:

<Tabs>
  <TabItem label="npm">
    ```sh
    npm install tegaki
    ```
  </TabItem>
  <TabItem label="bun">
    ```sh
    bun add tegaki
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh
    pnpm add tegaki
    ```
  </TabItem>
</Tabs>

## Generator

Use the [interactive generator](/generator/) on the Tegaki website to create font bundles from any Google Font. No additional packages need to be installed — the generator runs entirely in the browser.

## Requirements

- **Node.js** 18+ or **Bun** 1.0+
- **React** 18+ or 19
