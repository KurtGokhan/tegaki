## Renderer

Install the `tegaki` package to render handwriting animations in React:

<Tabs>
  <TabItem label="npm">
    ```sh
    npm install tegaki
    ```
  </TabItem>
  <TabItem label="bun">
    ```sh
    bun add tegaki
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh
    pnpm add tegaki
    ```
  </TabItem>
</Tabs>

## Generator

Install the `tegaki-generator` CLI to generate font data:

<Tabs>
  <TabItem label="npm">
    ```sh
    npm install -D tegaki-generator
    ```
  </TabItem>
  <TabItem label="bun">
    ```sh
    bun add -D tegaki-generator
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh
    pnpm add -D tegaki-generator
    ```
  </TabItem>
</Tabs>

## Requirements

- **Node.js** 18+ or **Bun** 1.0+
- **React** 18+ or 19
