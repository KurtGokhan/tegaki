<Steps>

1. **Generate font data**

   Use the [interactive generator](/generator/) to create stroke data for any Google Font. Configure options, then download the output bundle.

2. **Import the bundle**

   Add the downloaded bundle to your project and import it:

   ```tsx
   import bundle from './output/caveat/bundle';
   ```

3. **Render the animation**

   Use the `TegakiRenderer` component:

   ```tsx
   import { TegakiRenderer } from 'tegaki';

   function App() {
     return (
       <TegakiRenderer
         font={bundle}
         time={{ mode: 'uncontrolled', speed: 1, loop: true }}
         style={{ fontSize: 48 }}
       >
         Hello World
       </TegakiRenderer>
     );
   }
   ```

</Steps>
